== Description ==
This plugin add a custom post type for have a team archive and page for each member.

We do web development and if you need a developer or if you think you have found a bug in plugin, if you have any question, please feel free to contact us by this email info@13node.com.

== Installation ==

1. Upload the 'pro-teams' plugin folder to the '/wp-content/plugins/' directory.
2. Activate the "pro-teams" list plugin through the 'Plugins' menu in WordPress.
3. Ready, add some members.
